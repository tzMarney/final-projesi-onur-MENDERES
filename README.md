# Fitness İçerik Üretici

## Problem Tanımı

Sosyal medyada fitness içerikleri paylaşan kişiler sürekli yeni içerik fikirleri bulmak zorunda kalıyor. Bazen sadece video konusu bulmak bile zaman alabiliyor. Bu projede bu süreci biraz daha kolay hale getiren basit bir uygulama geliştirmeyi amaçladım.

## Hedef Kullanıcı

* Fitness içerikleri paylaşan kişiler
* Spor eğitmenleri
* Sporla ilgilenen sosyal medya kullanıcıları

## Projenin Kısa Açıklaması

Kullanıcı bir fitness konusu giriyor. Sistem de bu konuya göre video fikri, başlık, açıklama, hashtag ve kısa bir senaryo oluşturuyor.

## Kullanılan Teknolojiler

* Python
* Streamlit
* GitHub

## Sistemin Çalışma Mantığı

1. Kullanıcı bir konu girer.
2. İçerik oluştur butonuna basar.
3. Sistem konuya göre içerikler oluşturur.
4. Sonuçlar ekranda gösterilir.

## Kurulum

pip install streamlit

py -m streamlit run app.py

## Kullanım

Uygulama açıldıktan sonra fitness ile ilgili bir konu girilir. Daha sonra içerik oluştur butonuna basılarak sonuçlar görüntülenir.

## Testler

Uygulama aşağıdaki örnek konularla test edilmiştir:

* Göğüs antrenmanı
* Deadlift
* Yağ yakımı
* Bulk dönemi
* Protein tozu

## Eksik Yönler

Şu an oluşturulan içerikler hazır kalıplar kullanılarak üretilmektedir. Daha gelişmiş içerikler için yapay zeka entegrasyonu eklenebilir.

## İleride Yapılabilecekler

* Yapay zeka desteği eklenmesi
* Daha fazla içerik türü oluşturulması
* Farklı sosyal medya platformlarına göre içerik hazırlanması

## Yapay Zeka Kullanımı

Projenin planlama ve geliştirme sürecinde yapay zeka araçlarından yararlanılmıştır.
