# PROJE ÖNERİSİ

## Seçilen Görev
Seçenek 2 - Yapay Zeka Destekli Otomasyon

## Ürün Adı
Fitness İçerik Üretici (tıpkı benim gibi)

## Çözülecek Problem

Sosyal medyada fitness içerikleri paylaşan kişiler sürekli yeni video fikirleri bulmak zorunda kalıyor. Başlık, açıklama ve içerik fikri üretmek zaman aldığı için bu süreci kolaylaştıracak bir araç geliştirmek istiyorum.

## Hedef Kullanıcı

Fitness içerik üreticileri, sporla ilgilenen kişiler ve sosyal medya hesaplarını büyütmek isteyen kullanıcılar. (kendimi referans aldım)

## Kullanılacak Veri ve Kaynaklar

- Kullanıcının girdiği konu
- OpenAI API
- Yapay zeka tarafından oluşturulan içerikler

## Kullanılacak Teknolojiler

- Python
- Streamlit
- OpenAI API
- GitHub

## Beklenen Çıktı

Kullanıcı bir konu girdiğinde sistem:

- Video fikri oluşturacak
- Dikkat çekici bir giriş cümlesi verecek
- Başlık önerecek
- Açıklama yazacak
- Hashtag önerecek

## Diğer Çalışmalardan Farkı

Tek bir konu girilerek sosyal medya paylaşımı için gerekli olan birden fazla içeriğin aynı anda oluşturulması amaçlanmaktadır.
